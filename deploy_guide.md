# راهنمای نهایی دیپلوی روی Streamlit Cloud 🚀

چون سرویس Streamlit Cloud برای امنیت به اکانت شخصی شما متصل می‌شود، من نمی‌توانم مستقیماً دکمه "Deploy" را از طرف شما فشار دهم، اما تمام کارها را آماده کرده‌ام که در **۱ دقیقه** آن را نهایی کنید.

### گام ۱: اتصال گیت‌هاب به استریم‌لیت
۱. وارد سایت [share.streamlit.io](https://share.streamlit.io) شوید.
۲. دکمه **Create app** را بزنید.
۳. در بخش Repository، آدرس ریپوزیتوری خودتان را انتخاب کنید: `raminentezar-rgb/spl2`
۴. در بخش Main file path، آدرس فایل داشبورد را وارد کنید: `src/dashboard.py`

### گام ۲: تنظیمات امنیتی (خیلی مهم!)
قبل از زدن دکمه Deploy، روی **Advanced settings** کلیک کنید و در کادر **Secrets**، متن زیر را کپی و پیست کنید:

```toml
TELEGRAM_BOT_TOKEN = "8729090034:AAHUsKSw49XQai_SKSdP-P6c09-tEsf4avo"
TELEGRAM_CHAT_ID = "8336606408"

[trading]
data_source = "yahoo"
signal_only = true

[mt5]
login = 0
password = ""
server = ""
```

### گام ۳: پایان
دکمه **Deploy** را بزنید. 🏁

---
### چرا این روش عالی است؟
- **هزینه صفر**: این سرویس کاملاً رایگان است.
- **اجرای ۲۴ ساعته**: حتی اگر کامپیوتر و گوشی شما خاموش باشد، سرورهای استریم‌لیت بیدار می‌مانند.
- **آپدیت خودکار**: هر تغییری که من در کدهای شما (در گیت‌هاب) بدهم، استریم‌لیت به صورت خودکار خودش را آپدیت می‌کند.

**نکته درباره متاتریدر**: چون سرورهای استریم‌لیت لینوکسی هستند، نرم‌افزار متاتریدر روی آن‌ها نصب نمی‌شود. به همین دلیل من ربات را روی دیتای **Yahoo Finance** ست کردم تا همیشه و بدون خطا کار کند.
